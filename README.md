Clasificación Multiclase con Redes Neuronales

Este proyecto consiste en el entrenamiento y evaluación de tres modelos de redes neuronales (MLP) para un problema de clasificación multiclase utilizando datos tabulares.

Dataset

Se utilizó el dataset Wine, el cual contiene 178 muestras, 13 variables y 3 clases.

Modelos

Se implementaron tres modelos con diferente arquitectura:

Modelo 1: una capa oculta
Modelo 2: dos capas ocultas
Modelo 3: tres capas ocultas
Preprocesamiento

Se realizó escalamiento de datos y codificación one-hot.

Métricas

Se evaluaron los modelos utilizando:

Accuracy
Precision
Recall
F1-score
ROC-AUC
PR-AUC
Resultados

Se compararon los modelos para determinar cuál tiene mejor desempeño.

Conclusión

El modelo más complejo mostró mejor rendimiento, aunque puede presentar sobreajuste.
